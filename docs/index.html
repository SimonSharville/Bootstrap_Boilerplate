Bootstrap Boilerplate

I have created this boilerplate as a starting point for future projects.

The default bootstrap grid has been adjusted a little to reduce the gutter-width between columns, which I have always found to be a little wide. Details of this can be seen on the Structure page.

I have also created stylesheets for typography and a set of default Navbars. Components will be added as they are created and can be accessed through the UI Menu at the top.

The boilerplate has been constructed using CodeKit.
Feel free to fork it and use as you wish.

Simon Sharville
Graphic Designer

Note: Feel free to cut and paste any of the components coding, however, occasionally I have added ui-classes to certain elements. These are just for presentation purposes. ie. buttons have ui-btn on them to give them a little vertical space between them. So remove that ui-class when using it.
You should also remove the ui-styles.scss from core-styles.scss.
