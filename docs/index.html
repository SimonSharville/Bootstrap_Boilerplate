Bootstrap Boilerplate

I have created this boilerplate as a starting point for future projects.

The default bootstrap grid has been adjusted a little to reduce the gutter-width between columns, which I have always found to be a little wide. Details of this can be seen on the Structure page.

I have also created stylesheets for typography and a set of default Navbars. Other elements may be added as I create them.

The boilerplate has been constructed using CodeKit.
Feel free to fork it and use as you wish.
