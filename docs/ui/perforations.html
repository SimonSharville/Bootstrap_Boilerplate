Adjustments

Perforations & Dividers are designed to sit within the padding area of a column, so that they align with the text. If you wish them to push out of that and either fill a column or a row width then add the following classes.

Full width of a row

These have a class of full-row added to the perf or hr.

Full width of a column

These have a class of full-col added to the perf or hr.

Container Fluid Perforations & dividers

These both sit within a container-fluid to make them full width.

Breakout of a container

If you wish Perforations and HRs to remain within a container (and not a container-fluid), but want them to appear full width of the browser window, then add the class of breakout to them.

This also works with other components like panels.

Note: The breakout class need to be added to the component and not the container, row or column.

Vertical Rules between Columns

You can add a rule between columns by adding a class of vr to each of the columns.
This will only show on medium screens and above.

Note: this is not the same as Multi-column text as seen on the UI Typography pages.
