Tables within Panels

When placing a table within a Panel, you have the option of making the table flush to the sides of the panel or not. To make it flush ADD an extra class of .panel-table to the panel-body.

Default Table inside a Default Panel

This is a Table that sits within a Panel Body that has the added class of .panel-table. This forces the table to fit the full width of the panel.
This is an optional table caption.

Variable Column Widths

Bootstap tables spread flexible columns evenly across the table. Sometimes, for purposes of layout and legibility you may want to set a column or multiple columns to be a set width. To do this we have come up with Variable column widths for a table.

Simply add a class of width you require to the top column TH or TD and that column will set to that width, all others will adjust to fill. If you wish to have completely control the width of all columns then the widths must add up to 100
