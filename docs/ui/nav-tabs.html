Nav Tabs

This is a selction of Nav Tabs created for SARD. There are various layouts that can be used. Horizontal Tabs (default) and Vertical Tabs. Each can be nesed within the other, so it can get a little complicated.

Nav Tabs come in either Horizontal or Vertical format.
Either can be nested in the other.
Horizontal tabs can be standard width or justified.

Horizontal Tabs

This is the default style for Nav Tabs. It uses a class of sa-h-tabs.
So that the base bar of the tabs fills the width of the screen the sa-h-tabs should NOT sit within a container and row. The tab-content however should be in a container.
