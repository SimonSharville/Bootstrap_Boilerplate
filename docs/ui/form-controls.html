Form -controls

Everything should be contained within a fieldset and a field-group.
You can have multiple fieldsets within a form and multiple field-groups within a fieldset.

You can disable all controls within the fieldset at once, by adding disabled to it. (ie. <fieldset disabled>).

Note on Columns within a form-group: Some form-groups have columns nested within them. to make them align properly you can either add a row around these columns, or for drier coding, just add the class of row to the form-group class.

Validation states

Bootstrap includes validation styles for error, warning, and success states on form controls. To use, add .has-warning, .has-error, or .has-success to the parent element. Any .control-label, .form-control, and .help-block within that element will receive the validation styles.

Input groups

Extend form controls by adding text or buttons before, after, or on both sides of any text-based <input>. Use .input-group with an .input-group-addon or .input-group-btn to prepend or append elements to a single .form-control.

See Bootstrap Input Groups for further details
