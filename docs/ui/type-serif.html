Serif (Secondary style)

The sans serif font used here is Roboto Slab a free google font. This font only comes in 4 weights, below are the styles for the weights as well as examples of headings, paragraphs and other various type styles.

Roboto is the default font for this boilerplate, if you wish to use the serif font, then add a class of serif to the individual tag. If you wish to change from Serif to Sans-Serif globally, the swap the defaultFont and secondFont styles within typogaphy.scss

Multi-column text

To create a text block that spans multiple columns and reflows, you should wrap the text within a div that has a class of either multi-column-4, multi-column-3 or multi-column-2 depending on the maximum number of columns you wish to have at wider screen views.

Note: This is not supported in Internet Explorer 9 and earier.

This layout is built to demonstrate the use of multi-column text. The structure is based on This. and built using haml and scss as part of my experiments in responsive design.

When resized the layout will change from 4 columns down to 3, 2, then 1 and the article text will reflow within those columns.
