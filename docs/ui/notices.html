Notices

Notices are Lists with panels, they can either have a status added to them or not.

The status should be added to the li as a class, Available status' are:
status status-success
status status-warning
status status-alert
status status-danger
status status-none

Single line variations

Notices were intended to be two lines in each list item, you can just have a single line, however if you wish to have a status next to it, each list-item that is a single line should have a class of single added to it.

Small status symbols

Small status symbols have been designed for use on purely single line lists. Each status class should have a class of status-sm added to it, ie. status status-sm status-alert
