Note: To revert to the default Bootstrap setting, comment out the section GRID ADJUSTMENTS in global/structure.scss
You will also need to make adjustments in the megamenu.scss (overrides) to align it with the navbar.

Adjustments

Components are designed to sit within the padding area of a column, so that they align with the text. If you wish them to push out of that and either fill a column or a row width then add the following classes.

Full width of a row

These have a class of full-row added to the perf or hr.

Full width of a column

These have a class of full-col added to the perf or hr.

Container Fluid

Usually you would use Container-fluid to make something full width of the screen. However, occasionally you would want to keep a component within a container, but force it to breakout and appear as if it is in a container-fluid.

This first example is of Perforations within a container-fluid.

Breakout of a container

If you wish a component to remain within a container (and not a container-fluid), but want them to appear full width of the browser window, then add the class of breakout to them.

This also works with other components like panels.

Note: The breakout class need to be added to the component and not the container, row or column.
