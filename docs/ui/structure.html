Structure

Wrapper

When using a fixed navbar at the top and a fixed footer as in this example all the content of the site should be wrapped in a wrapper. This adds a margin top and bottom so that the content doesn't hide behind them. This can be adjusted in the structure.scss file.

Grid Structure

The Grid Structure has been amended slightly from the Bootstrap Default. We have adjusted the column-gutters to be 20px instead of the original 30px.

To revert to the default Bootstrap setting, comment out the section GRID ADJUSTMENTS in global/structure.scss
